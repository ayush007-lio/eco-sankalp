## Features

- Report litter and waste with images and location
- Admin dashboard to verify reports, manage queries and orders
- Learning modules with quizzes and progress tracking
- Green marketplace with cart, demo payment, order receipt & tracking
- Local persistence for demo flows (localStorage)
